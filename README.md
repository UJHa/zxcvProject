# zxcvProject
zxcv 키 사용 게임 제작

# 현재 조작법
    - 이동 : 방향키
    - 점프 : Space
    - 공격 : C

# 구현된 State
    - 기본(Idle)
    - 걷기(Walk)
    - 뛰기(Run)
    - 점프(Jump)
